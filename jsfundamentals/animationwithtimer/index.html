Sidecar Prototype

Sidecar is for site owners who have additional navigation and content that’s important to a subset of users, but not core to the primary experience of the page. Need to tuck login info, access to additional resources, or oft-used navigation that doesn’t fit with your core use cases?

Sidecar is designed to sit neatly offscreen with a small "nib" affordance that alerts site users to the presence of additional content and functionality.

For the prototype phase, Sidecar is built using jQuery as a dependency. Eventually we will want to make it more agnostic and not require a specific library. We use CSS transitions to animate the sidecar opening and closing. If you are using a browser that does not support CSS transitions you will simply see the Sidecar open and close without an animation.

Checkout Sidecar's GitHub page for installation instructions and examples.
