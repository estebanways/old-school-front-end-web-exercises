List of initialization properties

Common initialization options

  • parent - string or object, id of container, where tree will be initialized, mandatory
  • mode - tabbar orientation mode
  • height - height of tabbar's labels
  • image_path - {string} set path to image folder ( not affect already created element until their state changes )(setImagePath)
  • align - {string} set align of tabs on tabbar(setAlign)
  • auto_size - {boolean} enable / disable auto adjusting height and width to outer conteiner(enableAutoReSize)
  • close_button - {boolean} enable mode, in which each tab has close button, mode will be applied to the tabs created AFTER command(enableTabCloseButton)
  • content_zone - {boolean} enable/disable content zone (enabled by default)(enableContentZone)
  • forced - {boolean} (enableForceHiding)
  • margin - {integer} set distance between tabs(setMargin)
  • href_mode - {string} set mode of loading external content(setHrefMode)
  • offset - {integer} set offset before first tab on tabbar(setOffset)
  • scroll - {boolean} enable/disable scrollers ( enabled by default )(enableScroll)
  • size_by_content - {boolean} enable disable auto adjusting height and width to inner content(enableAutoSize)
  • skin - {string} set style used for tabbar(setSkin)
  • xml - {string} load tabbar from xml file(loadXML)

Events

any event can be assigned by using property equal to event name 

	tabbar = new dhtmlXTabBar({
		parent:"a_tabbar",
		onSelect:function(){
			alert("I'm selected");	
		}
	});
